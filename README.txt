Реализовать приложение для подсчета калорий.
 Пользователь вводит информацию об употребленных продуктах, приложение выдает ему количество полученных калорий.
 Обязательно реализовать механизм регистрации пользователя. При реги
 страции пользователь заполняет ник, пароль, рост и вес.

Выполняют Коковский Артём, Семержиди Яннис, Яценко Глеб.

Я (Артём) и Яннис выполняем обработку запросов пользователь. Глеб разрабатывает структуру регистрации.
